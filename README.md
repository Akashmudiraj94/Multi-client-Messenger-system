A lightweight real-time chat application developed in Java that allows multiple clients to communicate simultaneously with a central server on the same computer. The project demonstrates core networking concepts and multithreading. 
Key Features.
1.Multi-Client Support: Multiple users can connect to the server and exchange messages in real-time.
2.Simple GUI: User-friendly interface built with Java Swing or JavaFX.
3.Local Communication: Uses localhost and a specific port (e.g., 2020) for communication.
4.Multithreading: Handles concurrent client requests without blocking the server
